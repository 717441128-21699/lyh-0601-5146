# 在线编程竞赛平台 (Online Judge Platform)

大型在线编程竞赛的报名、比赛与智能评测系统，支持参赛者、竞赛管理员和评委三种角色。

---

## 🌟 功能特性

### 👤 三种角色体系
| 角色 | 功能说明 |
|------|---------|
| **参赛者** | 在线报名、选择竞赛组别、代码提交、查看实时排名、下载电子成绩单 |
| **竞赛管理员** | 设置竞赛规则与题目参数、配置防作弊阈值、管理用户、查看提交统计、人工复核作弊 |
| **评委** | 主观题在线打分、加权平均分自动汇总 |

### 🚀 核心业务模块

1. **智能报名与赛道分配**
   - 根据参赛者历史 Rating 成绩自动匹配竞赛组别
   - 按组别容量自动分配赛道，生成唯一参赛凭证
   - 支持手动报名和自动分配两种模式

2. **代码自动评测系统**
   - 支持 C++、Java、Python、JavaScript 四种主流语言
   - 多测试点编译运行，答案自动归一化比对
   - 时间限制、内存限制、运行时异常全面检测
   - Docker 沙箱隔离执行环境（可配置）

3. **实时排名系统**
   - 比赛期间实时更新个人排名
   - 按赛道分组排名 / 总排名
   - ICPC 计分规则（按解题数、罚时排序）

4. **反作弊检测**
   - 随机检查点代码相似度检测
   - SHA-256 精确哈希匹配 + Jaccard Token 结构相似度
   - 相似度超阈值自动标记，推送管理员人工复核

5. **主观题评委评分**
   - 多评委独立打分
   - 系统自动计算加权平均分
   - 评分评语记录

6. **电子成绩单与证书**
   - 比赛结束自动计算各赛道总成绩
   - 生成带排名和奖项的电子成绩单
   - PDF 格式下载

7. **消息实时推送**
   - 报名成功提醒
   - 成绩更新通知
   - 反作弊标记告警
   - 证书生成通知

8. **运营报表自动生成**
   - 每月 1 号自动统计各竞赛数据
   - 参与人数、平均得分、通过率等多维度分析
   - 推送到管理端查看

## 🚀 快速开始

### 环境要求

- **Node.js**: >= 18.0.0
- **MySQL**: >= 8.0
- **Redis**: >= 6.0（可选，用于缓存和实时推送）
- **Docker**: 可选，用于代码沙箱执行

### 步骤 1: 克隆并安装依赖

```bash
# Windows 用户可直接双击运行 setup.bat
setup.bat

# 或手动执行
npm install
```

### 步骤 2: 初始化数据库

1. 确保 MySQL 服务已启动
2. 修改 `backend/.env` 中的数据库连接信息
3. 执行初始化脚本：

```bash
mysql -u root -p < database/init.sql
# 或
npm run db:init
```

数据库初始化后会自动创建：
- 3 个管理员/评委账号
- 5 个示例参赛者账号
- 2 场示例竞赛
- 4 个竞赛组别
- 5 道示例题目
- 7 条示例报名记录

### 步骤 3: 启动服务

```bash
# Windows 用户双击 start.bat
# 或手动执行
npm run dev
```

服务启动后：
- 后端 API: http://localhost:3000
- 前端页面: http://localhost:5173

## 🧠 核心算法说明

### 智能赛道分配算法
1. 获取用户历史 Rating 分数
2. 遍历竞赛所有组别，匹配 Rating 区间（minRating ≤ rating ≤ maxRating）
3. 检查该组别是否还有容量
4. 优先选择精确匹配的组别；若满员则降级选择任意有空位的组别
5. 分配成功后更新组别当前人数，生成唯一参赛凭证编号

### 代码评测流程
1. **编译阶段**: 根据语言调用对应编译器（模拟或真实 Docker 环境）
2. **测试点执行**: 对每个测试点独立运行，记录时间/内存/输出
3. **答案比对**: 输出归一化（去换行/行尾空格）后与标准答案精确比对
4. **结果判定**: AC / WA / TLE / MLE / RE / CE / Cheating

### 反作弊检测
1. **SHA-256 哈希匹配**: 精确检测完全相同的代码
2. **Jaccard Token 相似度**: 分词后计算结构相似度
3. **相似度阈值**: 管理员可配置（默认 0.85）
4. **分级告警**: LOW / MEDIUM / HIGH / CRITICAL 四级

### 定时任务
使用 `@nestjs/schedule` 实现：
- **每月 1 号 00:00**: 自动生成上月运营报表
- **比赛结束后 5 分钟**: 自动生成最终排名和电子成绩单
- **每 5 分钟**: 扫描进行中比赛，触发随机反作弊检查点
